学习的方向

  • Java后端学习方向 ：
    1. 学习Java基础 ：
      • Java基础语法
      • 集合
      • I/O
      • 多线程
      • JDBC
      • 网络编程
    2. 数据库学习 ：
      • MySQL语法 ：DDL DML DQL
      • JDBC API
      • 数据库四大特性
      • 数据库连接池
    3. Web基础学习 ：
      • Http/TCP协议
      • servlet
      • Filter
      • Listener
      • Web容器(tomcat)
      • JSP
      • Session
      • Cookie
    4. Web主流框架 ：
      • Spring ：Bean容器 IOC AOP
      • SpringMVC
      • Mybatis
    5. Web框架进阶 ：
      • SpringBoot
      • SpringCloud
    6. 其他部分 ：
      • Redis
      • Git
      • Maven
      • Linux
      • Docker
